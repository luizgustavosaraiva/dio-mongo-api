# MongoDB API

Exemplo de aplicação .net core com mongodb.

Para teste:

https://localhost:5001/infectado

### Json de exemplo de objeto abaixo

```json
{
	"dataNascimento": "1990-09-21",
	"sexo": "M",
	"latitude": -20.9363018,
	"longitude": -48.477349
}
```

OBS: é preciso ter configurar o acesso ao banco no appsettings.json.
